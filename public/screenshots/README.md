## Image Requirements

- **Format**: PNG or JPG
- **Size**: Recommended 1200x800px or similar aspect ratio
- **Quality**: High resolution for portfolio display
- **Optimization**: Compress images for web performance

## How to Add Screenshots

1. Save your screenshots with the exact filenames listed above
2. Place them in this `/public/screenshots/` directory
3. The portfolio will automatically display them in the project cards
